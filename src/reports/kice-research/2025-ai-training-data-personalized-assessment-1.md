## 요약

자동채점 선행연구 분석을 바탕으로 교과별 인공지능 학습용 데이터를 수집해 자동채점 모델 개발의 기반을 마련한 1차 연도 연구다.

## 연구 배경 및 목적

인공지능 모델 개발의 성패는 학습용 데이터의 질에 의해 좌우된다. 본 연구는 자동채점에 관한 선행연구 분석 결과를 바탕으로 교과별 인공지능 학습용 데이터를 수집하고, 이를 활용해 인공지능 기반 자동채점 모델 개발의 기반을 마련하는 것을 목적으로 고품질 학습용 데이터 구축에 역량을 집중했다.

## 연구 방법

학교 현장의 서·논술형 평가에 적용 가능한 자동채점 모델 학습용 데이터 구축을 평가도구 설계, 응답 데이터 수집, 채점 라벨링의 세 단계로 진행했다. 평가도구 개발 단계에서는 국어·사회·수학·과학 교과별로 실제 교육과정 성취기준에 기반한 서술형 또는 논술형 평가도구를 개발하고, 각 문항과 연계된 채점 기준표는 평가 요소별 수행 수준을 여러 단계로 구분해 객관적·체계적 채점을 지원하도록 설계했다.

응답 데이터 수집 단계에서는 개발한 교과별 평가도구를 활용해 전국 단위 표집으로 실제 중·고등학생의 다양한 응답 데이터를 수집했다. 채점 라벨링 단계에서는 사전에 체계적인 채점자 워크숍을 통해 훈련된 교과별 현직 교사들이 수집된 학생 응답 데이터 전체에 대해 개발된 채점 기준표에 따라 일관성 있는 라벨링을 수행했다.

## 주요 결과

학교 현장에서 실제 활용되는 교과별 서·논술형 평가도구의 특성을 심층적으로 이해하고 이를 반영해 평가도구를 개발했으며, 인공지능 모델 학습에 최적화된 데이터 구조를 고려해 학생 응답 데이터와 채점 데이터를 유기적으로 결합한 학습용 데이터를 구축했다. 훈련된 현직 교사에 의한 일관성 있는 채점 라벨링을 통해 고품질 라벨링 데이터를 확보함으로써, 후속 연도의 자동채점 인공지능 모델 개발과 고도화로 이어지는 기반을 마련했다.
